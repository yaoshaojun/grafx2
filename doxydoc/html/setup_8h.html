void Set_config_directory ( const char *  program_dir,
char *  config_dir 
)

Determine which directory should store the user's configuration.

For most Unix and Windows platforms: If a config file already exists in program_dir, it will return it in priority (Useful for development, and possibly for upgrading from DOS version) If the standard directory doesn't exist yet, this function will attempt to create it (/.grafx2, or APPDATA%\GrafX2) If it cannot be created, this function will return the executable's own directory. IN: The directory containing the executable OUT: Write into config_dir. Trailing / or \ is kept.
